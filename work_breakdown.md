## 推奨進め方

```
フェーズ1 → 2 → 3-1〜3-2 → 4-1 → 5-3（シングルGPU版）→ 残りを追加
```

まずはシングルGPU・小データで「学習ループが1エポック回る」状態を目指すのが最短ルートです。DDP（分散学習）やEMAは最後に追加することをお勧めします。
